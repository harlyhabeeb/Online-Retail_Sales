# Retail Sales Analysis Project

## Project Overview
This project analyzes retail sales data to identify trends in customer behavior and product performance. Using Python, we explored monthly sales, top-selling products, and customer patterns.

## Data
- **Columns**: InvoiceNo, StockCode, Description, Quantity, InvoiceDate, UnitPrice, CustomerID, Country.
- **Source**: [https://archive.ics.uci.edu/dataset/352/online+retail].

## Key Insights
- **Monthly Sales Trends**: Sales peaked during holiday months.
- **Top Products**: Certain items consistently drive high revenue.
- **Customer Analysis**: Segment customers based on purchase behavior.
